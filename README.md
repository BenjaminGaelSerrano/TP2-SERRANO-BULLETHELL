# Bullet Hell Espacial Serrano 🚀

Mové tu nave, destruí meteoritos y matá al cometa antes de que se escape.

## Controles
- **← →** o **A D** para moverse
- **Espacio** para disparar
- **Escape** para pausar

## Cómo funciona
Destruís 50 meteoritos y aparece el Gran Cometa. Tenés 10 segundos para bajarle las 50 vidas antes de que desaparezca. Si perdés las 3 vidas, game over.
